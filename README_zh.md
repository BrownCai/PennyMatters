## 1. 核心功能

PennyMatters 采用对话式交互界面革新记账体验——如同与专属财务顾问发消息。目标用户为需要精准消费管控的人群。

**特色功能:**  
- 🗨️ **对话式记账**: 通过自然对话记录支出（示例："星巴克咖啡 3.5 欧元"）  
- 📸 **小票识别**: 上传购物凭证自动提取交易数据  
- 📊 **智能分析**: 获取周/月消费报告、价格波动提醒及可视化图表  
- 🛒 **比价功能**: 购物前查询本地商家实时价格基准
